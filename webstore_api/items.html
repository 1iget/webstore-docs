Web Store API:Items

Resource representation

JSON

{
  "kind": "chromewebstore#item",
  "id": string,
  "publicKey": string,
  "uploadState": string,
  "itemError": [
    (value)
  ]
}
Property name Value Description
id string Unique ID of the item.
itemError[] list Detail human-readable status of the operation, in English only. Same error messages are displayed when you upload your app to the Chrome Web Store.
kind string Item in the Chrome Web Store.
publicKey string Public key of this item.
uploadState string Status of the operation.

Acceptable values are:
  • "FAILURE":
  • "IN_PROGRESS":
  • "NOT_FOUND":
  • "SUCCESS":

Methods

get
Gets your own Chrome Web Store item.
insert
Inserts a new item.
publish
Publishes an item. Please provide defined publishTarget in the URL, either publishTarget="trustedTesters" or publishTarget="default" (case sensitive).
update
Updates an existing item.
