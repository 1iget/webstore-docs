Items: Publish

Publishes an item.

Request

HTTP request

POST https://www.googleapis.com/chromewebstore/v1.1/items/itemId/publish

Parameters

Parameter name Value Description
Path parameters
itemId string The ID of the item to publish.
Optional query parameters
publishTarget string The target audience of this publish action. Either "trustedTesters" or "default".

Request body

In the request body, supply data with the following structure:

JSON

{
  "target": string
}
Property name Value Description
target string The publish target of this publish operation.

Response

If successful, this method returns a response body with the following structure:

JSON

{
  "kind": "chromewebstore#item",
  "item_id": string,
  "status": [
    string
  ],
  "statusDetail": [
    string
  ]
}
Property name Value Description
kind string This response is about a Chrome Web Store item.
item_id string The id of this item.
status[] list The status of this publish operation. It may contain multiple elements.
statusDetail[] list Detailed human-readable explanation to the status.
