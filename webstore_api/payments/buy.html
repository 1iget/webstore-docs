Payments:Buy

Inserts a cart and returns the JWT.

Request

HTTP request

POST https://www.googleapis.com/chromewebstore/v1.1/payments/buy

Request body

In the request body, supply data with the following structure:

{
  "itemId": string,
  "sku": string,
  "hl": string,
  "gl": string
}
Property name Value Description
itemId string Unique identifier for a Chrome App, Chrome Extension or Chrome Theme.
sku string Unique identifer (Product ID) for an individual In-App Product (IAP). Product IDs must start with a lower-case letter or number and can only contain lower-case letters(a-z), numbers(0-9), dots(.) and underscores(_).
hl string Specifies the language code of the In-App Product (IAP) when projection is THIN.
gl string Specifies the region code of the In-App Product (IAP) when projection is THIN.

Authorization

This request requires authorization with the following scope.

Scope
https://www.googleapis.com/auth/chromewebstore
For more details on API authorization and authentication, please consult the OAuth 2.0 documentation.

Response

If successful, this method returns a response body with the following structure:

{
  "kind": "chromewebstore#jwt",
  "jwt": string
}
Property name Value Description
kind string Always a static string value of "chromewebstore#jwt"
jwt string JSON web token. Consumed by the Chrome Script and passed to Google Wallet.
