Payments:Buy

Inserts a cart and returns the JWT.

Request

HTTP request

POST https://www.googleapis.com/chromewebstore/v1.1/payments/buy

Request body

In the request body, supply data with the following structure:

{
  "itemId": string,
  "sku": string,
  "hl": string,
  "gl": string
}
Property name Value Description
itemId string The ID of the item to query for in-app products.
sku string The in-app product ID.
hl string Specifies the language code of the IAP when projection is THIN.
gl string Specifies the region code of the IAP when projection is THIN.

Authorization

This request requires authorization with the following scope.

Scope
https://www.googleapis.com/auth/chromewebstore.readonly

Response

If successful, this method returns a response body with the following structure:

{
  "kind": "chromewebstore#jwt",
  "jwt": string
}
Property name Value Description
kind string
jwt string JSON web token. Consumed by the Chrome Script and passed to the Google Wallet.
