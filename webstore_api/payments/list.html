Payments:List

Lists the in-app products that the user has purchased.

Request

HTTP request

GET https://www.googleapis.com/chromewebstore/v1.1/items/itemId/payments

Authorization

This request requires authorization with the following scope.

Scope
https://www.googleapis.com/auth/chromewebstore.readonly
For more details on API authorization and authentication, please consult the OAuth 2.0 documentation.

Request body

Do not supply a request body with this method.

Response

If successful, this method returns a response body with the following structure:

[
  {
    "kind": "chromewebstore#payment",
    "itemId": string,
    "sku": string,
    "createdTime": long,
    "state": string
  }
]
Property name Value Description
kind string Static string value. Always "chromewebstore#payment"
itemId string The ID of the item to query for in-app products.
sku string The in-app product ID.
createdTime long Time of the creation of the payment.
state string Will be one of the following payment states: [ACTIVE, PAYMENT_DECLINED, EXPIRED, CANCELLED, REJECTED, PENDING, CANCELLED_BY_DEVELOPER, DISABLED]
